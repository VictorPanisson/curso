• Lista 01
    • Questão 01- Esse exercício mostra como usar JavaScript para criar um campo onde o usuário pode digitar seu nome e, ao clicar em um botão, ver uma mensagem de saudação personalizada com o nome que digitou.
    • Questão 02- Este código cria um botão que, ao ser clicado, aumenta um contador de cliques exibido na página. O contador é inicializado em 0 e é incrementado a cada clique no botão.
    • Questão 03- Um formulário simples onde o usuário insere o ano de nascimento. Ao clicar no botão "Calcular Idade", a idade é calculada e exibida na página.
    • Questão 04- Verifica se a idade inserida pelo usuário é maior ou igual a 18 anos ao clicar em um botão, exibindo uma mensagem correspondente.
    • Questão 05- Dois botões para alternar a visibilidade de um elemento na página usando a propriedade style.display.
    • Questão 06- Um botão que, ao ser clicado, altera aleatoriamente a cor de fundo da página usando document.body.style.backgroundColor.
    • Questão 07- Uma calculadora que permite adicionar, subtrair, multiplicar e dividir dois números. Utiliza campos de entrada para números, um seletor para escolher a operação e um botão para calcular.
    • Questão 08- Conta quantas palavras estão em um texto grande inserido pelo usuário em um campo de texto (textarea), ativado por um botão de contagem.
    • Questão 09- Um relógio digital na página que atualiza a cada segundo, mostrando apenas o horário atual.
